Quiver
Arrow Flight · gRPC · Columnar Transport

Low latency, feature serving for ML inference.

Fetch features from multiple backends in a single call. Quiver handles fan-out, merging, and returns columnar data with Apache Arrow Flight.

Get started Quickstart
<1ms overhead per request
Zero-copy arrow delivery to clients
5+ Backend supported
inference.py
from quiver import Client   client = Client("localhost:8815")   table = client.get_features( feature_view="user_features", entities=["user:1000", "user:1001"], features=["score", "country"], )   # zero-copy Arrow → tensor tensor = table.to_torch()
