Arrow Flight · gRPC · Zero-copy columnar


Low latency, arrow-native feature serving.

Resolve features across Redis, PostgreSQL, S3, and ClickHouse in a single gRPC call. Zero-copy columnar transport via Apache Arrow Flight — built for ML inference pipelines.

Get started Quickstart
<1ms sidecar latency

0-copy columnar transport
Rust core
inference.py
from quiver import Client
 
client = Client("localhost:8815")
 
# fan-out across Redis + Postgres
table = client.get_features(
    feature_view="user_features",
    entities=["user:1000", "user:1001"],
    features=["score", "country"],
)
 
# zero-copy Arrow → tensor
tensor = table.to_torch()
